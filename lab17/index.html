OOP Design

Understand the Problem

  • set up the game
  • create the gallows
  • generate a word
  • create blanks for each letter in the word
  • get a letter guess from player
  • if the letter is in the word add it anywhere it applys in the letter blanks
  • if the letter is not in the word add an appendage to the end of the gallows
  • repeat until the game ends

Pseudocode the High Level Approach

set up the game

create game board

create gallows

create man

generate word

create blanks for every letter in the word

start game

ask player to guess a letter

input letter

if the letter is in the word add it anywhere it applys in the letter blanks

if the letter is not in the word reveal an appendage of the man

repeat until either the word is complete or the man is completley revealed

display player has won if the word was completed

display player has lost if the man was completed

repeat until either the word is complete or the man is completley revealed [player]

display player has won if the word was completed [game]

display player has lost if the man was completed [game]
